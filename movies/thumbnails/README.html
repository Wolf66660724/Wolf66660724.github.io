视频封面图片目录

请将视频封面图片（缩略图）放在这个目录下。

支持格式

  • .jpg - 推荐使用
  • .png - 支持透明背景

推荐尺寸

  • 1280x720 (16:9)
  • 1920x1080 (16:9)

文件命名

封面图片文件名应与对应的视频文件名一致（扩展名不同）。

例如：

  • 视频：我的视频.mp4
  • 封面：我的视频.jpg

文件路径示例

source/movies/thumbnails/我的视频.jpg
