# 文章图片目录

此目录用于存放博客文章的图片资源，包括文章封面图和内容图片。

## 📁 目录结构

每篇文章的图片应放在以文章标题命名的子文件夹中：

```
img/posts/
├── 记录生活的开始/
│   ├── cover.jpg      # 文章封面图（用于归档页面和首页显示）
│   ├── top-bg.jpg     # 文章顶部背景图（可选）
│   ├── image1.jpg     # 文章内容图片
│   └── image2.png
├── 这是小红帽说不清第几次踏上这条路了/
│   ├── cover.jpg      # 封面图
│   └── story-image.jpg
└── ...
```

## 📝 使用方法

### 1. 文章封面图（Cover）

封面图用于在归档页面和首页显示文章缩略图。

**在文章 front-matter 中设置：**

```yaml
---
title: 我的文章标题
date: 2025-11-20
cover: /img/posts/我的文章标题/cover.jpg
---
```

**封面图选项：**
- 图片路径：`cover: /img/posts/文章标题/cover.jpg`
- 颜色值：`cover: '#49B1F5'`
- 禁用封面：`cover: false`

**封面图建议：**
- 尺寸：1200x600 像素（2:1 比例）
- 格式：JPG（文件更小）或 PNG（支持透明）
- 文件大小：建议控制在 200KB 以内
- 命名：统一命名为 `cover.jpg`

### 2. 文章内容图片

在文章 Markdown 文件中引用图片：

```markdown
![图片描述](/img/posts/记录生活的开始/example.jpg)
```

**建议：**
- 图片宽度建议不超过 1200 像素
- 使用描述性的文件名
- 为图片添加有意义的 `alt` 文本

### 3. 文章顶部背景图（可选）

如果需要为文章页面设置独特的顶部背景图：

```yaml
---
title: 我的文章标题
top_img: /img/posts/我的文章标题/top-bg.jpg
---
```

如果不设置 `top_img`，会使用封面图或默认背景。

## ✅ 命名建议

- **文件夹名称**：使用文章标题（与 `_posts/` 中的文件名对应）
- **封面图**：统一命名为 `cover.jpg`
- **内容图片**：使用小写字母、数字和连字符
- **避免**：空格和特殊字符（中文可以使用）
- **使用描述性的文件名**：便于管理和查找

## 💡 完整示例

假设你有一篇文章 `source/_posts/我的第一篇文章.md`：

**1. 创建文件夹：**
```
source/img/posts/我的第一篇文章/
```

**2. 添加图片：**
- `cover.jpg` - 封面图
- `image1.jpg` - 内容图片1
- `image2.png` - 内容图片2

**3. 在文章中使用：**

```yaml
---
title: 我的第一篇文章
date: 2025-11-20 12:00:00
tags:
  - 技术
  - 教程
categories:
  - 技术分享
cover: /img/posts/我的第一篇文章/cover.jpg
---

## 文章内容

这是文章的正文内容。

![示例图片1](/img/posts/我的第一篇文章/image1.jpg)

![示例图片2](/img/posts/我的第一篇文章/image2.png)
```

## 📌 注意事项

- **归档页面显示**：封面图会在归档页面以缩略图形式显示（需在 `_config.butterfly.yml` 中启用 `cover.archives_enable: true`）
- **首页显示**：封面图也会在首页文章卡片中显示
- **路径一致性**：确保文件夹名称与文章标题完全一致
- **相对路径**：在 Markdown 中使用绝对路径（以 `/` 开头）引用图片
- **图片优化**：建议压缩图片以提升加载速度
