# 音乐封面图目录

此目录用于存放音乐播放器的封面图片。

## 📁 文件结构

```
img/music/covers/
├── song1-cover.jpg    # 歌曲1的封面图
├── song2-cover.jpg    # 歌曲2的封面图
└── ...
```

## 📝 使用方法

在音乐播放器配置中引用封面图：

```yaml
# 在音乐播放器配置文件中
- name: 歌曲名称
  artist: 艺术家名称
  url: /music/songs/song.mp3
  cover: /img/music/covers/song-cover.jpg
```

## 💡 说明

- 音乐文件放在 `source/music/songs/` 目录
- 封面图统一放在 `img/music/covers/` 目录
- 建议封面图尺寸：300x300 像素（正方形）
- 推荐使用 JPG 格式以减小文件大小
- 封面图文件名建议与歌曲文件名对应，便于管理

## ✅ 最佳实践

1. **命名规范**：使用歌曲名称或ID作为文件名
2. **图片质量**：使用高质量的封面图以获得更好的视觉效果
3. **文件大小**：建议控制在 100KB 以内
4. **格式选择**：JPG 用于照片类封面，PNG 用于图标类封面
